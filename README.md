## Getting Started

Clone the project

```bash
  git clone https://github.com/adenugbamichael/books-v2
```

Go to the project directory

```bash
  cd client
  cd server
```

Install dependencies

```bash
  npm install
```

Fire the server/client

```bash
  npm run dev
```
